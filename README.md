
          Ratlou Retail Store — Website (WEDE5020 POE)
Static website for the Ratlou Retail Store — Part 3 (Functionality & SEO).

## Live site
`https://ST10491339.github.io/ratlou-retail/`

## Features
- Home, About, Merchandise, Enquiry, Contact and Thank You pages
- JavaScript: dynamic product grid, search & filter, lightbox modal, accordion
- Client-side form validation (enquiry & contact)
- Form "processing" simulated on Thankyou.html (submission details displayed)
- SEO: meta tags, canonical links, robots.txt and sitemap.xml
- Embedded Google Maps for store locations

## How to run locally
1. Open the project folder in VS Code.
2. Use Live Server (or open `index.html`) in the browser.
3. Forms use GET and will show submission results on `Thankyou.html`.

## Files of interest
- `CSS/style.css` — styles
- `JS/main.js` — all JavaScript functionality
- `sitemap.xml`, `robots.txt` — SEO files
- `README.md`, `CHANGELOG.md` — repository docs
